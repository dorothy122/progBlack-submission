- vue : javascript framework used to build interactive user interfaces (https://vuejs.org/guide/introduction.html 10/3)
- svelte: a frontend framework that uses javascript (https://umatechnology.org/what-is-svelte-framework-and-how-do-you-use-it/ 10/3)
- SVG: Scalable vector graphics.  uses maths equations. Used to make icons, logos, graphs, charts etc. (https://umatechnology.org/what-is-svelte-framework-and-how-do-you-use-it/ 10/3)
- django: python framework for making websites using python (https://www.w3schools.com/django/django_intro.php 10/3)
- AWS: amazon web services. A cloud service. (https://aws.amazon.com/what-is-aws/ 10/3)
- azure: microsofts cloud service.
